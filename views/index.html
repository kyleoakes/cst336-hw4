The Marriam-Webster Dictionary defines a logic gate as "a device (as in a computer) that outputs a signal when specified input conditions are met".

Basically, a logic gate is a component which accepts two inputs (or sometimes more, except for NOT which only accepts one input) and outputs a signal, either true or false, based on the type of gate and the input signals.

For example, an AND gate outputs a true signal when both inputs are true. An OR gate outputs a true signal when either or both of its inputs are true. A NOT gate outputs a signal only when its input is false.

Logic gates are composed of transistors and underly all of computer logic.
